# FloatPaste {{version}} 测试版使用说明

打包日期：{{date}}

## 启动方式

1. 解压 zip 包到任意本地目录。
2. 双击 `floatpaste.exe` 启动。
3. 首次启动后，应用会常驻系统托盘。

## 首次运行提示

- 当前版本没有代码签名，Windows 可能提示风险警告。
- 若系统阻止运行，请在确认来源可信后手动放行。
- 若无法启动，请先确认系统已安装 WebView2 运行时。

## 建议优先验证

1. 复制文本后是否自动入库。
2. 全局快捷键是否能唤起 Picker。
3. 通过方向键、数字键和回车选择并回贴历史项。
4. 托盘菜单中的资料库、设置与暂停监听是否正常。
