# FloatPaste {{version}} 测试版发布说明

发布日期：{{date}}

## 本版定位

- 本版本为熟人内测版，仅用于验证核心桌面链路
- 当前仅支持 Windows 10 / Windows 11 x64
- 当前未做代码签名，首次运行可能出现系统警告

## 本轮重点验证

- 文本复制后是否能自动入库
- 全局快捷键是否能稳定唤起 Picker
- Picker 中键盘选择与回贴是否稳定
- 托盘、暂停监听和资料库窗口是否工作正常

## 手动验证建议

1. 在任意应用中复制几段文本，确认 Manager 中可以看到记录。
2. 使用全局快捷键唤起 Picker，测试 `Up / Down / Enter / Esc / 1..9 / Tab`。
3. 在记事本、浏览器输入框、VS Code 中分别测试回贴。
4. 切换暂停监听后，再复制文本，确认不会继续入库。

## 已知限制

- 当前只支持文本剪贴项
- 不同桌面应用对 `Ctrl+V` 注入的兼容性仍在收口
- 若系统缺少 WebView2 运行时，程序可能无法正常启动
