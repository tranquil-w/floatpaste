# FloatPaste 0.1.0-beta.3 测试版发布说明

发布日期：2026-03-17

## 本版定位

- 本版本为 UI/UX  polish 版本，重点改进视觉一致性与主题系统
- 当前仅支持 Windows 10 / Windows 11 x64
- 当前未做代码签名，首次运行可能出现系统警告

## 本轮新增内容

### Catppuccin 主题系统

- 引入完整的 Catppuccin 配色方案
  - Latte（浅色主题）：温暖奶油白基调
  - Macchiato（深色主题）：深蓝紫基调
- Peach 暖色系作为强调色，贯穿收藏、成功状态、焦点环等元素
- 通过 CSS 变量实现运行时主题切换，过渡动画平滑

### macOS 风格视觉系统

- **极弱边框系统**：采用半透明 rgba 边框，亮度主题 12% / 深色主题 8% 不透明度
- **语义化阴影层级**：xs/sm/md/lg 四级阴影，统一使用 macOS 风格柔和阴影
- **统一圆角规范**：窗口 16px、面板 12px、控件 8px、标签 4px
- **卡片表面语义化**：区分 window-shell、panel-surface、control-surface、card-surface

### 无障碍与交互改进

- 添加 focus-visible 样式支持，键盘导航更清晰
- 支持 prefers-reduced-motion 媒体查询，尊重用户减少动画偏好
- 添加 skip-link 跳过导航链接
- 优化滚动条样式，与主题系统保持一致

### 代码结构优化

- 主题变量集中管理于 `src/index.css`
- Tailwind 配置扩展语义化颜色令牌
- Picker 窗口透明背景强制补丁，消除圆角尖角问题

## 手动验证建议

1. 在设置中切换浅色/深色主题，观察过渡动画是否平滑。
2. 检查 Manager 窗口边框、阴影是否符合 macOS 风格。
3. 验证 Picker 面板圆角是否正常，无透明尖角。
4. 使用键盘 Tab 导航，确认 focus-visible 样式可见。

## 已知限制

- 当前只支持文本剪贴项（图片/文件支持仍在开发中）
- 不同桌面应用对 `Ctrl+V` 注入的兼容性仍在收口
- 若系统缺少 WebView2 运行时，程序可能无法正常启动
